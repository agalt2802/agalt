##Progetto per esame ingegneria del software avanzata

####Requisiti funzionali:
Il server Express deve connettersi al client attraverso le socket.

Il server Express deve inviare richieste al client attraverso le socket.

Il client deve ricevere le richieste dal server Express attraverso le socket.

Il client deve elaborare le richieste ricevute dal server Express e inviare una risposta.

Il server Express deve ricevere la risposta dal client attraverso le socket.

Il server Express deve elaborare la risposta ricevuta dal client.

####Requisiti non funzionali:
Il server Express deve essere implementato utilizzando Node.js.

Le socket devono essere implementate utilizzando la libreria Socket.io.

Il server Express deve utilizzare un database per archiviare i dati.

Il server Express deve fornire API REST per accedere ai dati.

Il client deve essere implementato utilizzando Apache Cordova.

L'applicazione client deve essere cross-platform e funzionare su Android e browser.

La comunicazione tra server e client deve essere sicura e crittografata utilizzando HTTPS.

Il server Express deve accettare la connessione di un solo client alla volta

Il server Express deve essere testato e documentato.
